# Memorix | Your Smart Study Assistant

**Memorix** is an innovative web application designed to help high school students organize their study time by analyzing their strengths and weaknesses in different subjects to generate a personalized study schedule.

---

## Tech Stack

* **Languages**: HTML5, CSS3, and JavaScript (ES6+).
* **Assets**: Custom icons, web fonts, and favicons for an enhanced visual experience.
* **Hosting**: GitHub Pages.

---

## How to Run

### 1) Online Version
You can access the live project immediately here:
[https://jasseramir.github.io/memorix/](https://jasseramir.github.io/memorix/main/)

### 2) Local Execution
* Download or extract the project files.
* Open the folder using **VS Code**.
* Run `index.html` using the **Live Server** extension (or simply open `index.html` in any browser).

> **Note**: The project works offline, though some fonts or icons may require an internet connection if not stored locally.

---

## The Intelligence Behind Memorix

The project utilizes a **Priority Algorithm** that performs the following:
1.  **Data Analysis**: Analyzes student grades and inputs.
2.  **Targeting**: Identifies subjects that require more effort and time.
3.  **Generation**: Produces a smart study schedule that distributes time efficiently for personalized learning.
