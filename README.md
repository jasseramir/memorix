# Memorix | Your Smart Study Assistant

**Memorix** is an innovative web application designed to help high school students organize their study time by analyzing their strengths and weaknesses in different subjects to generate a personalized study schedule.[span_4](end_span)

---

## Tech Stack

* **[span_5](start_span)Languages**: HTML5, CSS3, and JavaScript (ES6+).[span_5](end_span)
* **[span_6](start_span)Assets**: Custom icons, web fonts, and favicons for an enhanced visual experience.[span_6](end_span)
* **[span_7](start_span)Hosting**: [GitHub Pages](https://jasseramir.github.io/memorix/)[span_7](end_span)

## The Intelligence Behind Memorix

[span_12](start_span)The project utilizes a **Priority Algorithm** that performs the following:[span_12](end_span)
1.  **[span_13](start_span)Data Analysis**: Analyzes student grades and inputs.[span_13](end_span)
2.  **[span_14](start_span)Targeting**: Identifies subjects that require more effort and time.[span_14](end_span)
3.  **[span_15](start_span)Generation**: Produces a smart study schedule that distributes time efficiently for personalized learning.[span_15](end_span)
